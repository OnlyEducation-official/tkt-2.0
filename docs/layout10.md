# HEADER SECTION MASTER PROMPT  
## Project: The Kerala Table — South Indian Restaurant Website

Act as a **senior frontend developer** and create a premium, pixel-perfect, responsive **Header / Navbar section** for a South Indian restaurant portfolio website named:

# The Kerala Table

The project is built using:

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide React

The website represents a Kerala-inspired South Indian restaurant focused on:

- Kerala culture
- Authentic South Indian food
- Banana leaf dining
- Warm hospitality
- Traditional values
- Premium restaurant experience
- Cultural identity and integrity

The header must feel modern, premium, clean, and culturally aligned with the brand.

---

# 1. Main Goal

Create a responsive website header that works perfectly on:

- Desktop
- Tablet
- Mobile

The header should be:

- Sticky
- Elegant
- Minimal
- Premium
- Easy to navigate
- Fully responsive
- Smoothly animated
- Consistent with the global design system

The header must use the colors, spacing, typography, and variables already defined in `global.css`.

Do not create random colors manually.

---

# 2. Important Project Rules

Before writing the header code, follow these files:

## `global.css`

Use all design tokens from this file.

Use existing CSS variables for:

- Background
- Text
- Primary color
- Secondary color
- Border
- Muted text
- Card background
- Accent color
- Radius
- Shadows

Do not hardcode random hex colors unless absolutely required.
